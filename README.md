## Project Overview
This Tableau Story presents lightning strike data through an interactive visual narrative.  
It highlights key patterns in the data using story points and dashboards designed to make trends easier to interpret.

## What the story covers
- lightning activity trends over time
- monthly and yearly comparisons
- geographic patterns
- key insights presented through Tableau Story points

## Tools Used
- Tableau Public
- CSV dataset
